How to Run the Application

Download the code using Source Tree
https://github.com/fareed543/Angular-Reactive-Forms

open the root directory and using cmd run command npm install
it will download the library files of angular

After that run command ng serve
it will run the application

For Deployment run command ng build
For Production Deployment run command ng build --prod
For sub folder deployment run commnad ng build --base-href="site.com/folder-name
